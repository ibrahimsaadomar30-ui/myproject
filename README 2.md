# Simple Interest Calculator

This project is a **Simple Interest Calculator** written in Bash.

## Description
The calculator computes simple interest based on the following inputs:
- **Principal (P):** The initial amount of money
- **Rate of Interest (R):** Annual interest rate in percentage
- **Time Period (T):** Time in years

The formula used:  
**SI = (P × R × T) / 100**

## How to Use
1. Open the terminal.
2. Run the script:
```bash
bash simple-interest.sh
